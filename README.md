# my Digital Business Card

使用 Next.js、Framer Motion 和 GSAP 構建。

## 技術棧

- **框架**: Next.js 14
- **動畫**: Framer Motion, GSAP
- **3D**: Three.js
- **樣式**: Tailwind CSS
- **語言**: TypeScript

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 構建生產版本
npm run build
```

## 部署到 GitHub Pages

```bash
# 構建並部署
npm run deploy
```

## 頁面結構

- `/` - 主頁面（Hero + Showcase）
- `/card000` - 數位名片頁面

## 自定義

1. 修改 `src/app/card000/page.tsx` 中的個人資訊
2. 更換 `public/images/default.png` 頭像圖片
